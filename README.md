# Corporate website design

This is a code bundle for Corporate website design. The original project is available at https://www.figma.com/design/e6WHBf0ZEOD5nw7d4r3DHQ/Corporate-website-design.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Browser console notes

- Some browser extensions (e.g., Sixpence/OpenLayer) may log resource load errors like `main.tsx:1 Failed to load resource: 404`.
- These are not build/runtime errors of the application when the app builds successfully and the dev server is working.
- Use an extension-free browser profile to verify clean app behavior.
